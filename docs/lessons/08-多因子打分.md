# 第 8 课：多因子打分

## 什么是因子？

因子就是影响股票表现的特征或指标。比如：

- 质量因子：ROE、毛利率、现金流
- 估值因子：PE、PB、股息率
- 动量因子：近期涨跌幅、换手率

## 为什么叫"多因子"？

因为单一指标很难全面评价一家公司。

- 只看 PE：可能买到垃圾公司
- 只看 ROE：可能买到估值过高的公司
- 只看涨幅：可能追涨杀跌

多因子打分就是把多个维度的信息合成一个综合得分。

## 加权平均

本项目的打分公式：

```
总分 = 质量分 × 0.45 + 估值分 × 0.35 + 动量分 × 0.20
```

权重可以根据你的投资风格调整：

- 价值投资：估值权重更高
- 成长投资：质量权重更高
- 趋势投资：动量权重更高

## 代码中的实现

```python
total = q * self.quality_weight + v * self.value_weight + m_score * self.momentum_weight
```

## 因子投资的重要提醒

1. **历史表现不代表未来**：过去 ROE 高不代表以后也高。
2. **避免过度拟合**：不要用太复杂的权重去拟合历史数据。
3. **关注可解释性**：每个因子都要有投资逻辑支撑。
