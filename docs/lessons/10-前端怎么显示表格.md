# 第 10 课：前端怎么显示表格

## 前端三件套

- **HTML**：网页的骨架（结构）
- **CSS**：网页的样式（美观）
- **JavaScript**：网页的行为（交互）

本项目用 **React + TypeScript + Tailwind CSS** 做前端。

## React 是什么？

React 是 Facebook 开发的库，让你用"组件"的方式搭建网页。

组件就像乐高积木，可以拼出复杂的界面。

## 一个简单的组件

```tsx
function Hello() {
  return (
    <div className="text-blue-600">
      Hello, Screener!
    </div>
  )
}
```

## 显示表格

```tsx
function StockTable({ items }) {
  return (
    <table className="w-full">
      <thead>
        <tr>
          <th>代码</th>
          <th>名称</th>
          <th>得分</th>
        </tr>
      </thead>
      <tbody>
        {items.map((item) => (
          <tr key={item.symbol}>
            <td>{item.symbol}</td>
            <td>{item.name}</td>
            <td>{item.total_score}</td>
          </tr>
        ))}
      </tbody>
    </table>
  )
}
```

## Tailwind CSS

Tailwind 是"工具类"CSS 框架，通过类名直接写样式：

```html
<div className="bg-white p-4 rounded-lg shadow">
  这是一个带圆角和阴影的白色卡片
</div>
```

常用类：

| 类名 | 作用 |
|------|------|
| `bg-white` | 白色背景 |
| `p-4` | 内边距 1rem |
| `m-2` | 外边距 0.5rem |
| `rounded-lg` | 大圆角 |
| `shadow` | 阴影 |
| `text-blue-600` | 蓝色文字 |
| `font-bold` | 粗体 |
