# 第 2 课：变量、类型和基本运算

## 什么是变量？

变量就是给一个值取名字，方便以后使用。

```python
stock_name = "平安银行"   # 字符串（文字）
stock_code = "000001"     # 字符串
roe = 12.5                # 浮点数（小数）
shares = 100              # 整数
is_profitable = True      # 布尔值（是/否）
```

## 常见数据类型

| 类型 | 例子 | 说明 |
|------|------|------|
| `str` | `"平安银行"` | 字符串，用引号包裹 |
| `int` | `100` | 整数 |
| `float` | `12.5` | 浮点数 |
| `bool` | `True` / `False` | 布尔值 |
| `None` | `None` | 空值，表示"没有" |

## 基本运算

```python
pe = 15.0
eps = 2.0
price = pe * eps          # 乘法
print(price)              # 30.0

debt_ratio = 60.0
asset = 100.0
debt = debt_ratio / 100 * asset   # 除法
print(debt)               # 60.0
```

## 类型转换

```python
roe_str = "12.5"
roe_float = float(roe_str)    # 字符串转小数
print(roe_float + 1.0)        # 13.5

shares = 1500
shares_str = str(shares)      # 整数转字符串
print("持有股数：" + shares_str)
```

## 格式化输出

```python
name = "贵州茅台"
pe = 28.5
print(f"{name} 的 PE 是 {pe:.2f}")   # 贵州茅台 的 PE 是 28.50
```

`f"..."` 叫 f-string，是 Python 里最好用的字符串格式化方式。

## 项目实战

在 `backend/data/akshare_provider.py` 里，你能看到大量这样的转换：

```python
def _to_float(value, scale: float = 1.0) -> float:
    if value is None:
        return None
    try:
        return float(value) / scale
    except (ValueError, TypeError):
        return None
```

它的作用是把 AkShare 返回的各种奇怪格式安全地转成 float。
