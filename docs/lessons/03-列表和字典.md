# 第 3 课：列表和字典

## 列表（List）

列表就是一堆按顺序排列的东西。

```python
stocks = ["000001", "000002", "600519"]
print(stocks[0])      # 000001
print(stocks[-1])     # 600519（最后一个）

stocks.append("000858")    # 加到最后
print(len(stocks))         # 4
```

## 字典（Dict）

字典是"键值对"，像查字典一样用关键字找内容。

```python
stock = {
    "symbol": "000001",
    "name": "平安银行",
    "pe": 8.5,
    "pb": 0.8,
}

print(stock["name"])     # 平安银行
print(stock.get("roe"))  # None（因为没有这个键）
```

用 `.get()` 比直接用 `[]` 安全，因为找不到时不会报错。

## 遍历

```python
stocks = [
    {"symbol": "000001", "name": "平安银行"},
    {"symbol": "600519", "name": "贵州茅台"},
]

for s in stocks:
    print(f"{s['symbol']} - {s['name']}")
```

输出：

```
000001 - 平安银行
600519 - 贵州茅台
```

## 列表推导式

一种简洁的写法：

```python
symbols = [s["symbol"] for s in stocks]
print(symbols)   # ['000001', '600519']
```
