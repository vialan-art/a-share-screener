# 第 6 课：ROE、PE、PB 是什么？

## ROE：净资产收益率

**公式**：

```
ROE = 净利润 / 净资产 × 100%
```

**人话解释**：

假设你开了一家奶茶店，投入了 100 万（净资产），一年赚了 20 万（净利润），那么你的 ROE 就是 20%。

ROE 衡量的是公司用股东的钱赚钱的能力。

**参考标准**：

| ROE 范围 | 大致含义 |
|----------|----------|
| > 20% | 优秀（如茅台） |
| 15% - 20% | 良好 |
| 10% - 15% | 一般 |
| < 10% | 较弱 |

## PE：市盈率

**公式**：

```
PE = 股价 / 每股收益
```

**人话解释**：

PE 表示"按照现在的盈利水平，多少年能回本"。

PE = 10 意味着：如果公司每年赚的钱不变，你需要 10 年收回投资。

**参考标准**：

| PE 范围 | 大致含义 |
|---------|----------|
| < 10 | 便宜（可能是价值股，也可能有问题） |
| 10 - 20 | 合理 |
| 20 - 40 | 偏贵（成长股常见） |
| > 40 | 很贵或亏损 |

## PB：市净率

**公式**：

```
PB = 股价 / 每股净资产
```

**人话解释**：

PB 表示"股价是公司净资产的多少倍"。

PB < 1 意味着股价低于账面净资产，可能便宜，也可能公司资产质量差。

**适用场景**：

- 银行、保险等重资产行业常用 PB
- 科技公司常用 PE 或 PS

## 三者的关系

```
ROE = 净利润 / 净资产
PB = 市值 / 净资产
PE = 市值 / 净利润

所以：PB = PE × ROE
```

## 项目实战

在 `backend/scoring/engine.py` 里：

- 质量分用到了 ROE
- 估值分用到了 PE 和 PB
- PE 越低越好，ROE 越高越好

## 小练习

1. 一只股票的股价是 100 元，每股收益 5 元，每股净资产 20 元，计算 PE、PB、ROE。
2. 你认为银行和科技公司，分别更适合用 PE 还是 PB 估值？
